# Spool Studio v2.0.1

Maintenance update for the v2.0 redesign.

## Changes

- Fixed transparent filament color detection and display.
- Improved color-name editing so recognized names immediately update the selected color and empty fields are only reset after leaving the field.
- Added Marble/Marmor color recognition as a warm stone-gray filament color.
- Fixed change detection for Color Name updates before saving to Spoolman.
- Refreshed dynamic Spoolman-backed lists for spool selection, brands, locations, and empty spool weight suggestions when opened.
- Kept fixed app-controlled lists such as Filament Type and Variant stable while editing.
